# Data and Calibration Manifest

This folder stores literature and experimental constraints used to calibrate
or bound the ReS2 sliding-ferroelectric model.

`literature_constraints.csv` is not raw experimental data. It is a manifest of
anchors used by the default parameter set. Rows marked `qualitative` should not
be used as quantitative fits. Replace those rows with DFT tables or measured
Raman/SHG/PL/PFM data before making predictive claims.
